Privacy Policy

Last updated: April 19, 2026

This privacy policy applies to the GSecurity Ad Shield browser extension developed by Goran Stambuk.

Data Collection

GSecurity Ad Shield does not collect, store, transmit, or share any personal data or browsing information. The extension operates entirely within your browser.

How the Extension Works

The extension uses the following browser APIs to block ads and trackers:

  • Declarative Net Request API - Blocks network requests matching predefined ad and tracker patterns. All filtering rules are bundled locally within the extension.
  • Content Scripts - Runs scripts on YouTube pages to suppress ad-related elements. These scripts operate locally in your browser and do not send any data externally.

Permissions

The extension requests the following permissions:

  • declarativeNetRequest - Required to block ad and tracker network requests using locally stored rules.
  • Host permissions - Required to apply ad blocking rules to web pages you visit. No data about your browsing is collected or transmitted.

Third-Party Services

The extension does not use any third-party analytics, tracking, or data collection services.

Data Storage

The extension does not store any user data locally or remotely. All filtering rules are static and bundled with the extension.

Changes to This Policy

If this privacy policy is updated, the changes will be posted on this page with an updated date.
